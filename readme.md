## MINCMS 基于YII 1.1.14 开发

### 环境要求 
	*   PHP 5.3.11 或更高版本
	*   Apache 或 Ngnix 需要打开rewrite支持 
	*   MYSQL 5.1 或更高版本

### 相关技术
    *   PHP 5.3 的相关知识，如命名空间、闭包等
    *   Composer
    *   Bootstrap 
    *   Yii 1.x 
### window下安装 [Git Bash](http://msysgit.github.io/)
